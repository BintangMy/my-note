# Intro Go

## 1. Hello World
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello")
}
```

## 2. Compile 
```
go build <file.go>
```

then run the executeable file

but if we are on development, we can do
```
go run <file.go>
```
